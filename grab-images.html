FF Image Grabber

For each FurnishedFinder listing below, open it in a new tab, paste the script in the browser console, and it will copy image URLs to your clipboard.

Step 1: Open each listing

Click each link below to open it. For each one, open DevTools (Cmd+Option+J) and paste the script from Step 2.

Listing 2: 1925 Spanish Cottage (Highland Park) Listing 3: Hollywood Hills Listing 4: Quiet Cottage (Eagle Rock) Listing 5: ModernHome (Glendale) Listing 6: Wisteria Pergola (Mid-City) Listing 7: Jade Place (West LA) Listing 8: 502 Rosemary Lane (Burbank) Listing 9: NoHo Bungalow

Step 2: Paste this in the browser console (Cmd+Option+J)

(() => {
  const id = location.pathname.match(/\/property\/(\d+)/)?.[1];
  const imgs = [...document.querySelectorAll('img')]
    .map(i => i.src || i.dataset.src)
    .filter(s => s && s.includes('_pdp_/' + id))
    .slice(0, 5);
  const result = JSON.stringify(imgs);
  navigator.clipboard.writeText(result);
  console.log('Copied ' + imgs.length + ' image URLs!');
  alert('Copied ' + imgs.length + ' image URLs to clipboard!\\n\\nPaste them in the textarea on the grabber page.');
})();
